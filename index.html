HTML - Hyper Text Markup Language - HTML is the main type of document used on the web

There are 3 major components of the web experiance - HTTP is the protocol that allows this to happen
1. Browser- this is the program that allows you to display files on the web
2. The Internet - The world's largest computer network
3. Servers - the Computers that host the web content

HTML allows you to create text - what you see, Markup - how you see it, References - images and videos and Links - paths to different web pages
